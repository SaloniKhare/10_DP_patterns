# Problem:
## Given a triangle array, return the minimum path sum from top to bottom.
## At each step, you may move to adjacent numbers on the row below.
---
# Example:
## [
##      [2],
##     [3,4],
##    [6,5,7],
##   [4,1,8,3]
 ]
# Output: 11  (2 + 3 + 5 + 1)
